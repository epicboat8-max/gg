ℹ️ How to Use

  1. Open a Website - Visit any website with images (manga, comics, photos, etc.)
  2. Select Language - Choose your target language from the dropdown above
  3. Click Start Translation - The extension scans for images on the page
  4. Wait for Processing - Loading spinners appear on images while processing
  5. View Translations - Translations automatically overlay on the images
  6. Clear Anytime - Click "Clear All" to remove translations and start over

❓ FAQ

Q: Why is the extension so simple?
A: We focus on core functionality - no bloat, no tracking, no ads.

Q: Does it work offline?
A: OCR works offline with Tesseract.js. Translation requires internet for LibreTranslate API.

Q: Is my data safe?
A: Yes! OCR processes locally on your computer. Only extracted text is sent to translation API.

Q: What languages are supported?
A: 10+ languages including Bengali, Hindi, Tamil, Spanish, French, German, and more.

Q: Can I use it with hover mode?
A: Yes! Hover mode shows translations only when you hover over images for a cleaner look.

⌨️ Tips

  • Use Hover Mode for a cleaner reading experience
  • Use Clear All to remove all overlays and start fresh
  • The floating button appears on every webpage for quick access
  • Settings are automatically saved between sessions
  • Works with images on any website (Instagram, Twitter, forums, etc.)
